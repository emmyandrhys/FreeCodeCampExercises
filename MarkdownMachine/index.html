Markdown Text Editor

Instructions:

  • Enter your Markdown text on the editor box.
  • Your text will appear in formatted in the Preview class.

Formatting Tips:

Headings

Add 1-6 # symbols before your heading text. The more hash symbols the smaller the heading to be

Bold Text

**before and after** or __before and after__ make the text bold.

Italics

*before and after* or _before and after_ italicize the text.

Strikethough

~~before and after~~ strikeout the text.

Bold & Italics

***before and after*** make the text and bolded and italicized.

Nesting Bold and Italics

Bold and italics **can be _nested_ by nesting the symbols.**

Quoting

An > before text will set it off from the rest.

Code

`Single backticks` will leave that text unformatted. ```Triple backticks``` form a code block.

Links

Wrap the link text in [brackets](followed by the url in parenthesis).

Unordered List

Place a - or a * in front of each item/line.

Ordered List

Precede each line with a number and period (1.).

Nested Lists

Count the number of spaces in the line above to determine the number of spaces you want to indent. Enter that many spaces before marking the list item (* or - or 1.).

Task Lists

Line starts with a space, then []. To mark task complete, use [x].

Emoji

Type :emojicode: with a colon in front of and behind the emoji code.

Paragraphs

Leave a blank line between lines of text to mark paragraphs.
